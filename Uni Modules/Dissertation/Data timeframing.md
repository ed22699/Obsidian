## Using Yearly Data
- Pros:
	- more stable
		- smooths seasonality 
		- avoids spurious correlations
	- better aligned with a 3-year horizon decision
	- captures:
		- sustained tre
- Cons: 
	- fewer points

## Using Quarterly Data
- Pros:
	- more observations
	- higher temporal resolution
- Cons: 
	- strong seasonality
		- overreact to seasonal cycles
