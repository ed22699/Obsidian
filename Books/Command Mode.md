| Command               | Effect                            | Repeat | Reverse |
| --------------------- | --------------------------------- | ------ | ------- |
| /pattern              | search for that pattern forwards  | n      | N       |
| ?pattern              | search for that pattern backwards | n      | N       |
| :s/target/replacement | substitution                      | &      | u       |
|                       |                                   |        |         |
