Also work in bash shell

| Command | Effect                       |
| ------- | ---------------------------- |
| `<C-h>` | Backspace equivilent         |
| `<C-w>` | Delete back one word         |
| `<C-u>` | Delete back to start of line |
|         |                              |
Other commands

| Command          | Effect                                     |
| ---------------- | ------------------------------------------ |
| `<C-[>`          | Go to normal mode                          |
| `<C-o>`          | Insert normal mode                         |
| `<C-r>=5*12<CR>` | Calculations using the expression register |
|                  |                                            |
